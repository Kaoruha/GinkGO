## 达成目标的必要改进

### P0: 修复引擎非确定性 Bug
当前最大障碍。4 倍年化波动使结果不可信。
可能原因：dict 迭代顺序、共享可变状态、成交价格计算随机性。

### P1: 多标的/多策略组合
单标的单策略无法同时满足双目标。需要：
- portfolio 支持多策略并行
- 或多个 portfolio 协同管理
- 不同标的的策略可以互补

### P2: 修复 Decimal/float 类型 Bug
影响范围：mean_reversion、dual_thrust、bollinger_breakout 等多个策略。
统一方案：所有 `bar.close/high/low/open` 返回 float。

### P3: 更多数据频率
当前仅有日线数据。分钟线数据可提供：
- 更多的交易信号
- 更精细的入场/出场时机
- 更高的胜率可能性

## 技术问题记录

| 问题 | 影响 | Workaround |
|------|------|-----------|
| file_service.update() data 不生效 | 无法更新组件源码 | 创建新组件 |
| 并行 backtest run 失败 | 多个 run 同时卡住 | 顺序执行 |
| Decimal + float 混用 | 策略运行时报 TypeError | float() 转换 |
| ratio_sizer 资金不足 | 高股价无法交易 | 选低价股 |
| bar_adjustment Decimal bug | 复权计算失败 | 暂不使用复权 |
| position_ratio_risk 过度拦截 | 所有订单被拒 | 降低 ratio 或不用 |
