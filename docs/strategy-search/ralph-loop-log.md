## 关键发现

1. **momentum 是唯一稳定工作的策略**：其他策略（mean_reversion、trend_follow、dual_thrust、bollinger_breakout 等）在数据库中的代码运行后产出 0 订单，可能是运行时依赖或逻辑问题
2. **601899.SH（紫金矿业）是最佳标的**：000001.SZ（平安银行）和 000858.SZ（五粮液）均亏损
3. **最佳参数组合**：lookback=10, threshold=0.022, ratio=0.5, loss_limit=10%
4. **胜率瓶颈在 46%**：momentum 策略在天级别回测中胜率上限约 46%，无法达到 50% 目标
5. **年化可达 11.31%**（RL02），超过 10% 目标

## 问题记录

1. **ratio_sizer 数据库同步**：file_service.update() 的 data 字段更新不生效。workaround：创建新组件
2. **大量策略 0 订单**：数据库中的策略代码可能有运行时依赖问题（如 bar_service 调用方式不匹配）
3. **并行 backtest run 不可靠**：shell 脚本中 `&` 并行会导致 run 命令不执行。需分开用 Bash background 模式
4. **position_ratio_risk 过度拦截**：max_position_ratio=0.3 导致所有订单被拒绝
5. **高股价问题**：600519.SH（茅台 ~1600 元）在 100k 资金下 ratio_sizer 无法下单（不足 100 股）
