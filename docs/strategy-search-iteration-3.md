## 关键发现

### 1. 买入持有优于所有主动策略
在强趋势股票（紫金矿业、赛力斯）上，简单的买入持有大幅优于所有测试的主动策略（MA交叉、动量、双推等）。这是中国A股强趋势行情的典型特征。

### 2. 稳定性取决于买入时机
- 紫金矿业从10.5起步一路上涨，99.7%时间净值高于初始资金
- 赛力斯经历过大幅回调（48.9%最大回撤），稳定性较低但年化更高

### 3. Volume决定资金利用率
- Volume太小 → 资金利用率低 → 年化收益低
- Volume太大 → 超出资金 → 订单被拒
- 需要根据股价合理设置：volume = floor(100000 / stock_price * 0.9)

### 4. 定投策略（stab_*）有效
浦发银行涨幅不大（+43%），但通过分批买入20次（buy=0.99, max=20），实现了74.8%稳定性和12%年化。

### 5. 主动策略表现不如买入持有
- MA5/20 紫金: 5.1% 年化（vol=100），资金利用率过低
- MA5/20 赛力斯: 5.1% 年化, 89% 稳定性（频繁交易损耗大）
- Momentum60 紫金: 9.6% 年化, 85.2% 稳定性（最接近的主动策略）
- MA20/60 紫金: 6.7% 年化, 94.5% 稳定性

## 回测系统问题

### CLI `ginkgo backtest run` 超时
3年回测耗时超过5分钟（300秒），导致批量测试脚本频繁超时。建议增加默认超时或使用worker方式执行。

### DB组件代码过期（已修复）
源码更新后 MFile.data 不同步。已在第2轮修复所有内置策略。

### `file_service.update()` 不更新 data 字段（已知bug）
CLI `ginkgo component edit --file` 报告成功但实际不更新。需直接用 file_crud().modify()。
